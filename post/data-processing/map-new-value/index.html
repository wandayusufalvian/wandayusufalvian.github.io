Transform Dataframe Values Using Dictionary

how to change value in pandas column with new value that is stored in dictionary?

let say i have Dataframe data_census. It has column education with values below:

HS-grad         10501
Some-college     7291
Bachelors        5355
Masters          1723
Assoc-voc        1382
11th             1175
Assoc-acdm       1067
10th              933
7th-8th           646
Prof-school       576
9th               514
12th              433
Doctorate         413
5th-6th           333
1st-4th           168
Preschool          51
Name: education, dtype: int64

i want to change those categorical value as numerical using this dictionary :

di={
    'Doctorate':15,
    'Masters':14,
    'Bachelors':13, 
    'Assoc-voc':12, 
    'Assoc-acdm':12,
    'Some-college':12,
    'Prof-school':12,
    'HS-grad':11,
    '12th':10, 
    '11th':9,
    '10th':8,
    '9th':7, 
    '7th-8th':6,
    '5th-6th':5,
    '1st-4th':4,
    'Preschool':3
}

i can to that by using this code :

data_census['education']=data_census['education'].map(di)

now we can see the result using data_census['education'].value_counts() :

11    10501
12    10316
13     5355
14     1723
9      1175
8       933
6       646
7       514
10      433
15      413
5       333
4       168
3        51
