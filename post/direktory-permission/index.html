Directory Permission di Linux

Di Linux, tiap directory dan file memiliki permission untuk admin, group, dan other.

Untuk melihat permission tsb, ketik di terminal ⇒ ls -l , maka akan muncul directory beserta directory permission-nya.

(base) yusuf@DESKTOP-R0DO23T:~$ ls -l
total 8
drwxr-xr-x 26 yusuf yusuf 4096 Jan 10 22:27 anaconda3
drwxr-xr-x  2 yusuf yusuf 4096 Jan 24 16:00 belajar_shell
-rw-r--r--  1 yusuf yusuf    0 Feb  7 16:08 coba.txt

directory permission terdiri dari 4 bagian :

bagian 1 : d atau - ⇒ d adalah direktori dan - adalah file

bagian 2 : rwx (jika kosong,nilai = - ) ⇒ permission untuk admin

bagian 3 : rwx (jika kosong,nilai = - ) ⇒ permission untuk group

bagian 4 : rwx (jika kosong,nilai = - ) ⇒ permission untuk other

r= read

w=write

x=execute

jika ingin ubah permission admin :

chmod +rwx [nama file] //tambah permission rwx 
chmod -rwx [nama file] //kurangi permission rwx 
chmod +r [nama file] //tambah permission r 
dst...

jika ingin ubah permission group atau other tinggal tambah huruf g untuk group dan o untuk other :

chmod g+rwx [nama file]
chmod o+rwx [nama file]
