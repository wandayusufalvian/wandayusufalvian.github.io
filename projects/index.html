Database Benchmark in Raspberry Pi 3 [Github]

Raspberry Pi 3 is using 32 bit OS, has limited RAM, and has quite slow IO. Using this device as server to store time series data and retrieve that data through Web API is quite challenging. In this research, i benchmark several NoSQL database to store and retrieve time series data. Turn out that to store and retrieve 86400 data, using plain text json is the best option.
